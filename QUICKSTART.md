# TanglishScript Quick Start

## Get Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the CLI
```bash
node tanglish-cli.js --help
```

### 3. Build examples
```bash
npm run build
npm run build:complete
npm run build:style
```

### 4. Run examples
```bash
npm run run:app
npm run run:complete
```

### 5. Check syntax
```bash
npm run check
```

### 6. Open the IDE
```bash
npm run ide
```
Visit: http://localhost:4200

### 7. Try the learning portal
```bash
npm run learn
```
Visit: http://localhost:4300
